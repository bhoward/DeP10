DeP/10

This simulator was built by Brian Howard, with help from Jessica Jordan, to explore potential extensions to the Pep/10 system for use in upper-level classes such as Compilers and Operating Systems.

Built with Java, plus the following libraries:

  • RSyntaxTextArea
  • JCommander
  • FlatLaf
  • JTerminal

Github: https://github.com/bhoward/DeP10
