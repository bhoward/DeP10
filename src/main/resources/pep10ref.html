Special Memory Locations

  • FFF7,FFF8: Trap handler
  • FFF9,FFFA: Boot address
  • FFFB,FFFC: Initial stack pointer
  • FFFD: Character input (charIn)
  • FFFE: Character output (charOut)
  • FFFF: Power off (pwrOff)

Operation Cycle

PC ← Mem[FFF9]
SP ← Mem[FFFB]
do
    IR[0:7] ← Mem[PC], PX ← 0
    PC ← PC + 1
	if IR[0:7] is prefix
	    PX ← IR[0:7]
		IR[0:7] ← Mem[PC]
		PC ← PC + 1
    if PX,IR[0:7] is dyadic
        IR[8:23] ← Mem[PC]
        PC ← PC + 2
    Execute the instruction in PX,IR
while not power off && instruction was legal

Directives

.ASCII "string"
insert the ASCII character code bytes for the string
.BLOCK n
insert a block of n zero bytes
.BYTE x
insert a single byte given by x (low byte if x is a word)
label: .EQUATE x
associate the label with the value x in the symbol table
.WORD x
insert a two-byte word given by x
The following are intended for OS-level code
.ALIGN 1|2|4|8
advance the current position to a multiple of the given number
.DEFMACRO name[, nargs]
define a macro @name with the given number of arguments ($1, $2, ...)
.ENDMACRO
terminate a macro definition
.EXPORT sym
add the symbol to the export list
.INCLUDE "file"
assemble the contents of the given file at the current location
.INPUT sym
mark the symbol as memory-mapped input
.ORG addr
shift this section so that the given address is the current position
.OUTPUT sym
mark the symbol as memory-mapped output
.SCALL sym
generate a system call macro for the given symbol
.SECTION "name", "permissions"
start a named section with the given permissions (read/write/execute/zero)
